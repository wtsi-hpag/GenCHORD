\doxysubsection{Detailed Description}
\subsubsection*{template$<$\mbox{\hyperlink{structJSL_1_1NameValuePair}{class}} \mbox{\hyperlink{structJSL_1_1NameValuePair}{T}}$>$\newline
class JSL\+::\+Argument$<$ T $>$}
A class which allows arbitrary template parameters to be read in as command-\/line arguments or from a configuraiton file using a Name/\+Value pair system. Upon construction, the \doxylink{classJSL_1_1Argument_a83ada5bfa412192f76dd4290f679defd}{Value} parameter takes the default value (if provided), until it is overriden by a successful argument/\doxylink{classJSL_1_1ArgumentInterface_a29460ecfd033ad1a485333767e28b816}{Trigger\+String} match.
