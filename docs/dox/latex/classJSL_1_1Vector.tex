\doxysubsection{Detailed Description}
A class implementing basic R\texorpdfstring{$^\wedge$}{\string^}n vector mathematics. Mostly acts as an extention to the basic std\+::vector object, but with the implicit assumption that the objects should behave like members of a true vector space, can unambiguously overload some operators and add in additional functionality.
